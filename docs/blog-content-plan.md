# Beit Building Blog — 8-Post Evergreen Content Plan

This is the content roadmap for the second wave of blog posts after the Phase-10 flagship (`/blog/florida-hurricane-roof-insurance-claim-guide`). Each post in this batch is evergreen — meaning the content stays useful for years with light annual updates, not breaking-news content that decays.

Posts ship one at a time. Each draft cycle: outline → approve → draft → photo source → publish → analytics check before drafting the next. Cadence target: one post per 1-2 weeks.

---

## Prioritization criteria

Order is set by leverage, not by how interesting the topic is to write:

1. **Commercial intent** — does the searcher want to hire someone, or just learn? Hire-intent ranks first.
2. **Search volume** — broader queries pull more traffic, all else equal.
3. **Difficulty** — can a small contractor site realistically rank top-10, or do Angi / HomeAdvisor / Forbes lock the SERP?
4. **Internal-link strength** — does this post anchor to / from existing pages (city pages, flagship, future posts)?
5. **Seasonal timing** — hurricane prep is timed to peak June 1; shed when out of season.
